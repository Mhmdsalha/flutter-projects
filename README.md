# Flutter | Flutter Projects

A curated collection of Flutter applications gathered in one repository for portfolio, learning, and reference purposes. The repository includes UI practice projects, state-management exercises, and larger application samples such as news, e-commerce, and social apps.

## Projects Included

| Project | Folder | Summary |
| --- | --- | --- |
| BMI Calculator | `bmi_calculator` | A simple body mass index calculator app built for practicing layouts and user input flows. |
| Counter Screen | `counter_screen` | A basic Flutter counter-style project focused on state updates and widget structure. |
| Flutter Project 1 | `flutter_project1` | An early practice project covering Flutter fundamentals and screen composition. |
| Login Screen | `login_screen` | A login UI project for form layout, validation patterns, and clean screen design. |
| Messenger Design | `messenger_design` | A messaging interface mockup inspired by chat application layouts. |
| News App | `news_app` | A categorized news browsing app with search and API-driven content. |
| Shop App | `shop` | An e-commerce Flutter application with product flows and shopping features. |
| Social App | `social_app` | A social networking app with user interaction features and a larger app structure. |
| Todo App | `todo` | A task tracking app for practicing CRUD-style flows and local app structure. |
| User Screen | `user_screen` | A UI-focused user profile or account screen project. |

## How To Run Any Project

1. Open the desired project folder.
2. Install dependencies:

```bash
flutter pub get
```

3. Start the application:

```bash
flutter run
```

## Notes

- This repository is intended as a consolidated Flutter portfolio and practice archive.
- Some projects are simple UI exercises, while others are more complete application samples.
- The `social_app` project preserves parts of its original internal package naming from the source project.
- API keys and sensitive local runtime files should not be committed when extending these projects.
